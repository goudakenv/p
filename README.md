# ポートフォリオサイト

## 概要
このサイトは、自分のスキルや経歴をまとめたポートフォリオです。  
HTML・CSS・JavaScriptを使って作成しました。  

現在は静的サイトですが、今後Firebaseを使ったログイン機能やデータ管理機能の実装を予定しています。

- 自己紹介ページ
- スキル一覧表示
- 簡単なお問い合わせフォーム（現在は未実装）
